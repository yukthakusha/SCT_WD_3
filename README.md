# Tic Tac Toe Web Application

This project is a Tic Tac Toe web application developed as **Task 3** for the **SkillCraft Technology Web Development Internship**.

## Features
- Player vs Player gameplay
- Displays winner message
- Detects draw condition
- Tracks game state
- Restart game functionality
- Interactive and user-friendly UI

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Purpose
This task demonstrates:
- Handling user click events
- Game state management
- Winning condition logic
- Interactive web application development
